# WEATHER FIX V3

## 수정 내용

1. 초단기실황 수정
   - `getUltraSrtNcst` 사용
   - 매시 40분 이전에는 직전 시각 자료로 자동 조회
   - 조회 실패 시 최대 5시간 전까지 재시도

2. 초단기예보 수정
   - `getUltraSrtFcst` 사용
   - HH30 기준시각 적용
   - 45분 이전에는 직전 발표자료로 자동 조회
   - 시간순 정렬 후 6개 시간 표시

3. 중기예보 수정
   - 기존 `getMidFcst` 단독 조회 제거
   - `getMidLandFcst` + `getMidTa` 결합 방식으로 변경
   - 육상예보 코드: `11F20000` 광주·전남
   - 기온예보 코드: `11F20401` 여수
   - 오전/오후 날씨, 강수확률, 최저/최고기온 함께 표시

4. 상세보기 표시 수정
   - 중기예보 제목/설명 문구 수정
   - 최저/최고기온 표시 추가
   - 잘못된 AWS 번호 표시 제거

5. 샘플데이터 대체 방지
   - 초단기예보와 중기예보는 실제 API 결과만 표시
   - 실패 시 조회 결과 없음으로 표시
