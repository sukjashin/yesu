# 공공데이터포털 전용 수정본 v2

## 수정 내용
- APIHub 관련 호출 없이 공공데이터포털 방식만 사용합니다.
- 특보 조회 시 serviceKey가 이중 인코딩되어 실패할 수 있는 문제를 수정했습니다.
- 특보 조회 결과가 없거나 문구를 만들 수 없을 때 화면에 `내용없음`을 표시하도록 수정했습니다.
- 특보 API 오류 시에도 화면에는 `내용없음`으로 표시되도록 프론트 표시 로직을 정리했습니다.
- 초단기예보/중기예보 조회 시 일부 행사장 실패가 전체 실패로 번지는 문제를 완화했습니다.
- 상세보기 팝업은 API 결과가 없으면 빨간 에러 대신 빈 결과 안내 표를 표시합니다.

## 중요
- `.env`의 `KMA_SERVICE_KEY`에는 공공데이터포털에서 발급받은 단기예보/중기예보/예보통보문 권한이 있는 일반 인증키를 넣어야 합니다.
- 키가 APIHub 키이거나 해당 서비스 활용 신청이 안 되어 있으면 날씨 데이터는 나오지 않습니다.
